# Random VTuber Toys

A collection of chaotic tools for VTube Studio models.

## Toys

### [GetDown](GetDown/)

Generates chaotic random movements for your VTuber model. Every parameter gets randomized every frame — face, eyes, mouth, body, everything.

Available as a **standalone Python script** or a **Streamer.bot extension**.

## VTube Studio Setup

1. Open VTube Studio
2. Go to Settings > General Settings
3. Enable "Start API" (note your port number)
